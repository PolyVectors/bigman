# bigman
A Little Man Computer (LMC) Virtual Machine that handles compilation from Assembly to machine code and emulation of the CPU and RAM all while prioritising speed.

# features
- [x] Working compiler
- [ ] Working VM
- [ ] CLI
- [ ] Error reporting
